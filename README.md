# s2http

Convert socks proxy to a http proxy

# Prerequisite

A socks proxy(probably a client) is listening

It may not directly connect to a socks server

# Usage

```
Usage of ./s2http:
  -port string
        port to listen (default "8080")
  -q    silent the output
  -socks string
        socks url (default "127.0.0.1:1081")
  -v    show version.
```

end.
